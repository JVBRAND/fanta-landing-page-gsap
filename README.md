# Fanta Landing Page com GSAP

Fiz esse projeto acompanhando um tutorial, com o objetivo de praticar animações com GSAP e melhorar meu entendimento em HTML, CSS e JavaScript.

A ideia foi montar uma landing page mais visual, com animações acontecendo conforme o usuário vai descendo a página.

Deploy:
https://fanta-gsap-joao.netlify.app/

Tecnologias usadas:
- HTML
- CSS
- JavaScript
- GSAP (ScrollTrigger)

O que eu trabalhei nesse projeto:
- animações com scroll usando GSAP
- posicionamento de elementos na tela
- organização do layout
- integração do JavaScript com o visual da página

O que aprendi:
- como controlar animações com base no scroll
- como pensar melhor na estrutura do layout
- como deixar a página mais dinâmica usando JS

Projeto feito como prática para reforçar os conceitos.
